# Network Layout Expansion

## Overview
This project demonstrates how to expand a small business network to support additional staff and wireless connectivity. It was completed as part of the Microsoft IT Support Specialist Professional Certificate.

## Objectives
- Identify additional networking equipment.
- Design an expanded network layout.
- Identify network access layer protocols.
- Document the solution.

## Additional Equipment
- Router
- Network switch
- Wireless Access Point (WAP)
- Firewall
- Ethernet cables
- Desktop and laptop computers
- Network printer

## Network Access Layer Protocols
- Ethernet (IEEE 802.3)
- Wi-Fi (IEEE 802.11)

## Skills Demonstrated
- Network design
- TCP/IP fundamentals
- Network topology
- Wireless networking
- Technical documentation

## Learning Outcome
This exercise strengthened my understanding of network expansion, infrastructure planning, and the communication protocols used in modern business networks.
